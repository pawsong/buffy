# pasta

pasta is a platform where users can write and visualize code.

## Features

- Code editor
- Compiler
- Native modules
- 3D renderer

## Prerequisite

### GraphicsMagick or ImageMagick

```bash
brew install imagemagick
brew install graphicsmagick
```

### MongoDB

## Local Setup

```bash
# Project root
$ npm install

# `modules/compiler`
$ npm install
$ npm start

# `modules/game-srv-zone`
$ npm install
$ npm start

# `modules/console`
$ npm install
$ npm start
```
