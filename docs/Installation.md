# Installation

## Prerequisite

### node.js

node >=4 (Recommend using [nvm](https://github.com/creationix/nvm))

### GraphicsMagick or ImageMagick

```bash
brew install imagemagick
brew install graphicsmagick
```

### MongoDB

## Local Setup

```bash
# Project root
$ npm install

# `modules/compiler`
$ npm install
$ npm start

# `modules/game-srv-zone`
$ npm install
$ npm start

# `modules/console`
$ npm install
$ npm start
```
